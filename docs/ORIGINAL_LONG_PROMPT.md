# Project Context

This is a public educational web platform for children.

## Target users
- Children grades 0–4
- Children grades 5–8
- Parents manage child accounts

## Tech assumptions
- Use a modern web stack
- Keep implementation simple, secure, and maintainable
- Videos are embedded from private/unlisted YouTube links
- PowerPoint files should be displayed in a user-friendly web format

## Product rules
- Content is gradually added as courses
- Authentication is required
- Parent account controls child access
- Ami is an 11-year-old girl with brown hair and green eyes
- Moti is Ami’s orange cat

## Coding rules
- Ask before major architectural changes
- Prefer small commits
- Do not refactor unrelated files
- Before changing files, explain the intended change briefly
- After changes, provide test commands
